(1) First, download AirSim. AirSim can be downloaded here:

https://github.com/Microsoft/AirSim/

Click on "Clone or Download" (green right), then on "Download ZIP". Once
it is downloaded, you should unzip.


(2) Next, download an environment. AirSim environments can be downloaded here:

https://github.com/Microsoft/AirSim/releases

Then click on v1.2.2 - Windows, scroll down, and download an environment.
A good one to start with is one of the smaller ones, like Coastline.


(3) Information about how to use AirSim is available here:

https://github.com/Microsoft/AirSim


(4) If you want to get a feel for AirSim, run the "run.bat" file in the Coastline
folder.
